# SeePort

*SeePort*

SeePort is a personal entertainment tracking website that helps users organize movies, TV shows, and anime in one place.

Built using HTML, CSS, and JavaScript, the website provides a clean and responsive interface for browsing content, viewing detailed information, and managing watch statuses such as "To Watch," "Watching," and "Completed."

**Features**
- Browse Movies, TV Shows, and Anime
- Dedicated detail pages for each title
- Watch status organization system
- Responsive design for different screen sizes
- Modern and user-friendly interface
- Dynamic content handling with JavaScript
  
**Technologies Used**
- HTML5
- CSS3
- JavaScript

  
**Project Goal**

The goal of SeePort was to practice front-end web development by creating a complete multi-page application with custom styling, interactive features, and a structured user experience.
